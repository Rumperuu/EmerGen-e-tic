## Description

CachingPolicy is an EmerGen(e)tic module for testing the use of genetic
algorithms as applied to the cache updating behaviour of a web server.

## Installation

Copy everything in this directory into your EmerGen(e)tic directory. Then,
run the Terminal command `chmod +x ./project/cachingpolicy/setup.sh`.
